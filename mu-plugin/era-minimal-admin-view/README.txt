Minimal WP Admin med Toggle

Installation:
1. Upload mappen "minimal-wp-admin-toggle" til `/wp-content/mu-plugins/` på din WordPress-installation.
2. Ingen aktivering er nødvendig – MU-plugins er automatisk aktive.

Funktion:
- Skjuler standardmenuer og dashboard-widgets for et minimalistisk interface.
- Beholder Custom Post Types, "Udseende"-menu og WP Help.
- Tilføjer en toggle-knap i admin baren: "Vis alle indstillinger" / "Skjul ekstra indstillinger".
- Valget gemmes pr. bruger.
